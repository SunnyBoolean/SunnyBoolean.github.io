java中一些重要的关键字

发表于 2018-10-31 | 分类于 Java基础

       
java中的关键字(保留字)接近50个，这里我们列出比较难懂但是又很关键常用的一些关键字：

final、finally、static、synchronized、transient、volatile

final

       关于final这个关键字，从字面意思就是永久的，固定不变的，在java中的使用率几乎100%。

使用场景

final主要用于修饰变量、方法、类，但是不能修饰代码块。

(3）修饰方法

      final修饰的方法是不能被子类重写的。所以final就可以起到方法被别人重写的可能。

(4) 修饰类

      final修饰类是无法被子类继承的。所以final就可以起到类被继承的可能，比如String类就被定义为final类型，所以我们无法重写String类。

(5)final的本质

      我们知道final修饰变量一旦被赋值后就不能再更改，那么是值变量的值不能被更改还是指它的引用不能被更改呢？实际上，final的变量就是告诉虚拟机，变量所持有的内存地址不能再修改了，但是变量所持有的内存地址如果是一个对象，那么对象实际上是可以任意修改的，如下例子是没毛病的：

综上所述，final修饰变量的本质是不能修改变量的值，如果是引用类型，引用类型的值就是对象的内存地址.

总结

  1. 全局变量如果不在定义的时候就初始化编译器会提示报错，而局部变量则不用在定义的时候初始化，他可以在后面赋值一次。
  2. 全局变量和局部变量都只能赋值一次。
  3. final修饰变量后，改变量的值不能再被更改，如果是引用类型的变量，则改变量保存的是对象的内存地址。
